# LP1_SAGEP

Projeto da Disciplina Linguagem de Programação I - SAGEP - Sistema de Automação e Gestão da Secretaria Paroquial
O projeto visa desenvolver um site para paróquias que desejem automatizar o processo de inscrição para preparação para sacramentos e para realização de encontros promovidos pela igreja. O sistema substituirá o processo manual, permitindo que os paroquianos se registrem através de formulários eletrônicos disponibilizados de modo  online (web).

Funcionalidades Essenciais:
Mural de Avisos: Exibição clara dos períodos de inscrição e datas dos Sacramentos, .Encontros e Cursos de Formação)
Formulários Online: Disponibilização de formulários digitais para cada atividade.
Controle de Vagas: O secretário paroquial definirá um limite de inscritos. O sistema deve bloquear novas inscrições diretas após esse limite ser atingido.
Lista de Espera: Ao atingir o limite, o sistema deve registrar os interessados seguintes em uma lista de reserva, para alocação em caso de desistência.
Notificações: Envio automático de e-mails de confirmação de inscrição e lembretes sobre as datas das atividades.
